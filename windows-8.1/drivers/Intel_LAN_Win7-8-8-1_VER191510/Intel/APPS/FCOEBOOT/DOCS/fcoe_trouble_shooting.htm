Troubleshooting

The table below lists problems that can possibly occur when using Intel FCoE. For each problem a possible cause and resolution are provided.

Problem

Resolution

FCoE does not load on system startup and the sign-on banner is not displayed.

  

DCB/FCoE is disabled if an Intel® 82599-based port is teamed with an adapter that does not support DCB/FCoE.

 Remove the DCB/FCoE adapter from the team.
